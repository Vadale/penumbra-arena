# penumbra-analytics

Real statistical analysis on the simulation stream.

## Concept taught

Each module teaches one work-grade technique:

- `descriptive.py` — summary stats via Polars; effect sizes and CIs.
- `inferential.py` — Mann-Whitney U, permutation, χ² hypothesis tests.
- `econometrics.py` — OLS + Newey-West HAC, VAR, Granger causality.
- `monte_carlo.py` — Sobol quasi-Monte-Carlo, bootstrap, VaR/CVaR.
- `linalg.py` — graph Laplacian, spectral embedding, Fiedler vector.

These five give the user enough hands-on coverage of the major
statistical-analysis pillars. Topology, optimal transport, causal
inference, survival, Bayesian, and topic modelling join in later
phases.
